# BanTracker
A simple discord bot that tracks bans on Hypixel using their API.

## Usage
Open config.json and insert your API Key(s), bot token, and the channel ID of the channel you want the bans to be logged in.

Example:
``` 
json
{
    "token": "BOT-TOKEN",
    "channel": 1234567890
}
```
